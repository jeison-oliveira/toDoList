# Descrição da Atividade

Nesta atividade, você deverá desenvolver um aplicativo To-Do List utilizando a biblioteca Redux
Toolkit. As principais funcionalidades a serem implementadas são: Inserção de Atividade,
Remoção de Atividade e Finalização de Atividade. Abaixo, estão ilustradas algumas figuras que
exemplificam como a interface deve ser projetada. É obrigatório a utilização do Redux, e a
estrutura do projeto deve seguir o padrão demonstrado em sala de aula. Para a captura de
entrada de dados do usuário (Input de Atividades), você pode utilizar o hook useState.

## Especificações:

Ao remover uma atividade, ela deve ser excluída da lista. Quando uma atividade for marcada
como finalizada (botão verde), ela deve ser visualmente diferenciada na interface, indicando que
foi concluída (Não deve ser excluída).

## Entrega:

Você deverá gravar um vídeo curto, com duração máxima de 4 minutos, demonstrando o
funcionamento do aplicativo desenvolvido. Além disso, explique como o código foi organizado e
desenvolvido. O código-fonte deve ser hospedado no github, e o link para acesso deve ser
disponibilizado no comentário da atividade.

# Começando

Estas instruções permitirão que você obtenha uma cópia do projeto e a execute em sua máquina local para fins de desenvolvimento e teste.

# Pré-requisitos

Este projeto foi construído usando Node.js versão v18.16.0 e npm versão 9.5.1.

# Instalação

- Clone o repositório:

```sh
git clone https://github.com/jeison-oliveira/toDoList.git
```

- Navegue até o diretório do projeto:

```sh
cd toDoList
```

- Instale as dependências do projeto:

```sh
npm install
```

# Executando a Aplicação

- Para executar a aplicação, use o seguinte comando:

```sh
npm start
```

## Acessando a Aplicação

- Para executar a aplicação frontend, acessar a seguinte url:

```sh
http://localhost:3000/
```
